사용 방법

② HTML 파일 드롭 — 작성한 보고서 HTML 파일을 업로드
③ 메타데이터 입력 — 제목, 날짜, 설명, 태그 입력
④ 등록 — 파일이 posts/에 저장되고 posts.json이 자동 업데이트
⑤ Git push — Vercel 자동 배포 완료
